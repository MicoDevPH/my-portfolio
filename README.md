# Mico L. Nakase — Portfolio

Personal portfolio website built with React and Vite.

## Stack

- **React 19** — UI
- **Vite** — build tool with HMR
- **Vanilla CSS** — custom properties for dark/light theming, no CSS framework

## Features

- Loading screen with MLN. logo that flies to the navbar on dismiss
- Hero entrance: MICO slides from left, NAKASE from right, silhouette pops in
- Scroll-driven hero exit animation
- Dark / light mode toggle with localStorage persistence
- Active nav link tracking via IntersectionObserver
- Responsive layout

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
